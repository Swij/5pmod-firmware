5PMOD nRF52840
###########

Overview
********

The nRF52840 breakout board with five full-size PMOD connectors and an addressable RGB LED.

.. figure:: img/5pmod_nrf52840.png
     :width: 442px
     :align: center
     :alt: 5PMOD nRF52840

     5PMOD nRF52840

Hardware
********

Connections and IOs
===================

LED
---

* LED0 (blue) = P1.00
* addressable LED (WS2812B) = P0.08

Push buttons
------------

* RESET = SW0 = boot/reset/P0.18
